# Simple C++ Calculator

## Description
This is a simple console-based calculator written in C++. It supports basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with error handling for division by zero)
